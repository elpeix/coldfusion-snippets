coldfusion-snippets
===================

Coldfusion snippets for Sublime Text

Type the snippet shortcode and then press <kbd>Tab</kbd> to complete the snippet.

---
__cfapplication__
__cfargument__
__cfcomponent__
__cfdump__
__cffunction__
__cftry__

```html
<cftry>
  $1

  <cfcatch type="$2">
    $3
  </cfcatch>
</cftry>
```
